# 420Clock.net

420Clock.net is a fun little website for forgetful stoners. 

# F.A.Q:

Q: What is it?
A: During normal operation, 420Clock is a lightweight dark-mode clock that displays the local time on your machine, with a twist: At 4:20 (and 4:22) the clock enters a special celebration mode to remind you to hit that shit. It stops by design at 4:21.

Q: Why did you make this?
A: Because it seemed like an obvious idea, it's dirt cheap, it's fun, and it didn't exist before I created it.

Q: Do you make money off of this?
A: No, I do not. I want to add a sponsored section soon, but as it stands right now I'm not monetizing 420clock.

Q: How does it work?
A: 420Clock is written in lightweight HTML, CSS, and JS. You deploy it through the deployment script and then upload the resulting folder to CF Workers or GitHub Pages, then hook up an external domain. Boom, you have 420Clock.
